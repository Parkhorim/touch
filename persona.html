2025-2026 터치교사단(정보교육학회) 자료

데이터로 읽는 우리 반 아이들

12가지 학생 페르소나 카드를 통해 다각적 데이터(로그, 관찰, 상호작용)를 분석하고,
최적의 AI 도구와 휴먼 터치 전략을 매칭해보세요.

TYPE A-1

무기력한 관찰자

#수학기초부족 #자존감저하

DATA EVIDENCE

  • 진단평가 하위 20%
  • 문제 풀이 시간 5분 미만 (포기)
  • 힌트 버튼 클릭 0회 (시도X)

TEACHER'S NOTE

"틀릴까 봐 아예 시작을 안 해요. '모르겠어요'가 습관이 되었습니다."

TYPE A-2

초반 과열 방전형

#지구력부족 #용두사미

DATA EVIDENCE

  • 시작 10분 활동량 상위 10%
  • 후반부 활동량 급격한 하락
  • 화면 전환 횟수 후반부 20회↑

TEACHER'S NOTE

"처음엔 눈이 초롱초롱한데, 심화 과제만 나오면 엎드리거나 딴짓을 해요."

TYPE A-3

게임 몰입형

#보상의존 #SpeedRun

DATA EVIDENCE

  • 배지/포인트 학급 1위
  • 텍스트 체류 시간 3초 미만
  • 오답 재시도율 낮음 (바로 넘김)

TEACHER'S NOTE

"내용은 안 읽고 버튼만 눌러요. '다 깼다!'고 외치지만 물어보면 몰라요."

TYPE A-4

AI 의존형

#Copy&Paste #사고력저하

DATA EVIDENCE

  • 과제 문장 수준 전문가급
  • 오프라인 작문 수행 불가
  • 입력 방식 대부분 '붙여넣기'

TEACHER'S NOTE

"GPT 답변을 그대로 긁어옵니다. '이 단어가 무슨 뜻이니?' 물으면 당황해요."

TYPE A-5 (NEW)

질문 폭격기

#과잉호기심 #수업단절

DATA EVIDENCE

  • 수업 중 발언/질문 상위 1% (빈번)
  • 검색 키워드 주제와 무관한 심화어
  • 진도 달성률 낮음 (딴길로 샘)

TEACHER'S NOTE

"호기심은 좋은데 '근데 만약에...'라며 수업 흐름을 자꾸 끊어요."

TYPE B-1

관계 지향 산만형

#투머치토커 #분위기메이커

DATA EVIDENCE

  • 채팅/댓글 수 압도적 1위
  • 개인 과제 시간 미달
  • 협업 도구 기여도 높음 (반응 위주)

TEACHER'S NOTE

"친구 반응 살피느라 공부를 못해요. 채팅창 도배의 주범입니다."

TYPE B-2

고립된 우등생

#협업거부 #독불장군

DATA EVIDENCE

  • 개인 성취도 전과목 만점
  • 동료 평가 점수 최하위
  • 공동 편집 기여 삭제/수정 독점

TEACHER'S NOTE

"'제가 다 할게요'라며 남의 글을 지워버려요. 조별 활동 때마다 민원이 들어옵니다."

TYPE B-3

보이지 않는 아이

#무색무취 #관심필요

DATA EVIDENCE

  • 성적/과제물 완벽한 평균
  • 교사 상담/메모 기록 없음
  • 발표/질문 횟수 0회

TEACHER'S NOTE

"사고도 안 치고, 아주 잘하지도 않아서... 솔직히 기억에 잘 안 남아요."

TYPE C-1

디지털 거부형

#아날로그선호 #기계치

DATA EVIDENCE

  • 기기 스크린타임 0에 수렴
  • 지필 평가 점수 상위권
  • 디지털 리터러시 기초 미달

TEACHER'S NOTE

"태블릿만 들면 얼어버려요. '종이로 하면 안 돼요?'라고 매번 묻습니다."

TYPE C-2

완벽주의 지연형

#강박증 #제출지연

DATA EVIDENCE

  • 임시 저장 횟수 50회 이상
  • 최종 제출 시각 마감 1분 전/지각
  • 백스페이스 키 입력 매우 빈번

TEACHER'S NOTE

"썼다 지웠다를 무한 반복해요. 퀄리티는 좋은데 제시간에 못 냅니다."

TYPE C-3

정서적 불안형

#감정기복 #가정/교우문제

DATA EVIDENCE

  • 감정 출석부 '우울/화남' 빈도↑
  • 보건실 방문 주 3회 이상
  • 학습 불참 시간 불규칙적 (엎드림)

TEACHER'S NOTE

"오늘 기분 어때? 가 먼저입니다. 작은 자극에도 예민하게 반응해요."

TYPE C-4 (NEW)

디지털 창작자

#딴짓대장 #금손능력자

DATA EVIDENCE

  • 영상/편집 앱 사용 압도적 1위
  • 멀티미디어 과제 전문가급 퀄리티
  • 타이핑 속도 매우 빠름 (실시간)

TEACHER'S NOTE

"수업 중에 몰래 유튜브 보거나 영상 편집해요. 근데 기술은 저보다 좋아요."
